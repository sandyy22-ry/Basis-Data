Contoh Kasus

Pada sistem kampus, data mahasiswa, dosen, dan mata kuliah perlu diatur dalam database yang saling terhubung.

Contoh Struktur Tabel:

CREATE TABLE Mahasiswa (
  NIM CHAR(10) PRIMARY KEY,
  Nama VARCHAR(50),
  Jurusan VARCHAR(30)
);

CREATE TABLE Dosen (
  NIP CHAR(10) PRIMARY KEY,
  Nama VARCHAR(50)
);

CREATE TABLE MataKuliah (
  KodeMK CHAR(6) PRIMARY KEY,
  NamaMK VARCHAR(50),
  SKS INT
);
