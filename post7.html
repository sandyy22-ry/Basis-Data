Perbedaan DDL dan DML

SQL terbagi menjadi beberapa jenis perintah, dua di antaranya adalah DDL dan DML.

DDL (Data Definition Language)

Digunakan untuk mendefinisikan struktur database, seperti:

  • CREATE — membuat database atau tabel baru
  • ALTER — mengubah struktur tabel
  • DROP — menghapus tabel atau database

DML (Data Manipulation Language)

Digunakan untuk memanipulasi isi data:

  • INSERT — menambah data baru
  • UPDATE — memperbarui data
  • DELETE — menghapus data
  • SELECT — menampilkan data
