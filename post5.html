Instalasi MySQL

langkah-langkah Instalasi MySQL di Windows

  • Download Installer Buka situs resmi: https://dev.mysql.com/downloads/installer/ Pilih MySQL Installer (Windows). Unduh versi MySQL Installer Full agar lengkap. Jalankan Installer Klik dua kali file .msi yang sudah diunduh. Pilih opsi instalasi: Developer Default (direkomendasikan) atau Server only jika hanya butuh database. Pilih Komponen MySQL Server MySQL Workbench MySQL Shell Connector (jika dibutuhkan) Proses Instalasi Klik Execute atau Install untuk memulai proses pengunduhan dan pemasangan. Konfigurasi Server Pilih tipe konfigurasi: Standalone Pilih port default: 3306 Pilih mode autentikasi: Use Strong Password Encryption Buat Password Buat password untuk user root. Opsi untuk menambah user lain (bisa dilewati dulu). Selesaikan Setup Klik Apply dan tunggu proses konfigurasi selesai. Klik Finish. Coba Jalankan Gunakan MySQL Workbench dan login dengan: User: root Password: (yang sudah dibuat sebelumnya)
