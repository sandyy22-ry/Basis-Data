Bahasa SQL Dasar

SQL (Structured Query Language) digunakan untuk mengakses dan mengelola data dalam sistem database relasional.

Perintah Dasar SQL

  • SELECT — Menampilkan data
  • INSERT — Menambah data
  • UPDATE — Mengubah data
  • DELETE — Menghapus data
