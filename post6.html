Pengenalan SQL

SQL (Structured Query Language) adalah bahasa standar untuk mengelola dan memanipulasi data pada sistem basis data relasional.

Perintah Dasar SQL:

-- Membuat tabel
CREATE TABLE Mahasiswa (
  NIM CHAR(10) PRIMARY KEY,
  Nama VARCHAR(50),
  Jurusan VARCHAR(30)
);

-- Menambah data
INSERT INTO Mahasiswa VALUES ('2401010001','Sandy Tirta','Informatika');

-- Menampilkan data
SELECT * FROM Mahasiswa;

-- Mengubah data
UPDATE Mahasiswa SET Nama='Sandy T.' WHERE NIM='2401010001';

-- Menghapus data
DELETE FROM Mahasiswa WHERE NIM='2401010001';
