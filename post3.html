Model Data

Model data menggambarkan bagaimana data diatur, dihubungkan, dan disimpan dalam basis data.

Jenis Model Data

  • Model Hierarki: Data disusun seperti pohon (parent-child).
  • Model Jaringan: Data saling terhubung banyak-ke-banyak.
  • Model Relasional: Data disimpan dalam tabel menggunakan relasi antar kolom.
