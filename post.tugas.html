Tugas Praktik SQL

Berikut daftar tugas untuk latihan SQL dasar:

  1. Buat database kampus di MySQL.
  2. Buat tabel mahasiswa dengan kolom NIM, Nama, dan Jurusan.
  3. Masukkan minimal 5 data mahasiswa.
  4. Tampilkan seluruh data mahasiswa menggunakan SELECT *.
  5. Perbarui nama salah satu mahasiswa menggunakan UPDATE.
  6. Hapus 1 data mahasiswa menggunakan DELETE.
  7. Simpan semua hasil query SQL di file tugas.sql.
