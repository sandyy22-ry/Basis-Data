Tugas Praktik SQL

  1. Buat database kampus di MySQL.
  2. Buat tabel mahasiswa dengan kolom NIM, Nama, Jurusan.
  3. Masukkan minimal 5 data mahasiswa.
  4. Tampilkan seluruh data mahasiswa dengan SELECT *.
  5. Ubah nama salah satu mahasiswa dengan UPDATE.
  6. Hapus satu data dengan DELETE.
